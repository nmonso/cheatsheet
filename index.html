Table Elements

Tag Elements Description
<table> Table Comprised of rows and columns of cells containing data
<thead> Table Header Headings of the table should be inside of this element. Also used to divide the sections of the table in HTML
<tbody> Table Body Table body should side inside of this element
<tfoot> Table Footer Table foot belongs inside this element
<th> Table Head Represents the heading for either a column or a row
<tr> Table Row Represents each table row
<td> Table Data Represents the data in each cell of a table
